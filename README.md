plenty-rest-python
===============================

version number: 0.0.1
author: Pierre Geier

Overview
--------

A python package that can be installed with pip.

Installation / Usage
--------------------

To install use pip:

    $ pip install plentyrest


Or clone the repo:

    $ git clone https://github.com/bloodywing/plentyrest.git
    $ python setup.py install
